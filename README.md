# Домашнее задание: Обработка запросов и шаблоны

Проект содержит два функциональных блока:

## 1. Рецепты
Сервис для отображения ингредиентов блюд.

Доступные маршруты:
- `/` — список доступных рецептов
- `/omlet/` — омлет
- `/pasta/` — макароны с сыром
- `/buter/` — бутерброд

Поддерживается параметр `?servings=N` для расчёта количества ингредиентов на N порций.

Пример:  
http://127.0.0.1:8000/omlet/?servings=4

## 2. Пагинация
Чтение CSV-файла с остановками наземного транспорта и постраничный вывод (10 записей на страницу).

Маршрут:
- `/bus_stations/` — список остановок с навигацией

Примеры:
- http://127.0.0.1:8000/bus_stations/
- http://127.0.0.1:8000/bus_stations/?page=2

## Запуск проекта

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

3. Откройте в браузере:
   - Главная: http://127.0.0.1:8000/
   - Остановки: http://127.0.0.1:8000/bus_stations/
