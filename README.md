# Домашнее задание: Обработка запросов и шаблоны

Проект содержит два функциональных блока:

## 1. Рецепты
Сервис для отображения ингредиентов блюд.

Доступные маршруты:
- `/` — список доступных рецептов
- `/omlet/` — омлет
- `/pasta/` — макароны с сыром
- `/buter/` — бутерброд

Поддерживается параметр `?servings=N` для расчёта количества ингредиентов на N порций.

Пример:  
http://127.0.0.1:8000/omlet/?servings=4

## 2. Пагинация (в разработке)
Чтение CSV-файла с остановками наземного транспорта и постраничный вывод.

## Запуск проекта

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

3. Откройте в браузере: http://127.0.0.1:8000/
